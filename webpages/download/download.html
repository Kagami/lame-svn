Download Page

Don't want to mess with patching entire directories?  Jan Peman's site now contains the complete LAME source distribution.

Linux & Unix:  Binaries and complete source available from the not_lame encoder site.

Mac: Source code and Mac specific patches available from the MacLame site.

Windows:  LAME .dll included with CDex.  The command line executable is available on  mp3tech.org as well as this site in Russia.

DOS and OS/2:  (and Windows, if you know how to use a command line interface) check out  (Not)LAME for OS/2 and DOS

For those of you that just wont give up on your Amiga, check The Amiga Alternative Audio Page

There is also the GOGO-no-coda  MP3 encoder.   They have forked LAME development at LAME 3.23beta.  Many key routines rewritten in assembly.  Requires the NASM assembler.

Takehiro Tominaga has done some amazing work on LAME recently (see  history).  His  versions are available here.    Fortunatly for us,  Takehiro takes the time to fold many of his improvements back into LAME.

This is a source code patch against the ISO demonstration source. It contains none of the original ISO encoder source code.  It  includes the LAME encoder patches, mpg123 for decoding, the GPSYCHO psy-model and the MP3x graphical analyzer.
  • Requires ISO demonstration source: dist10.tar (FhG, MP3Tech)
  • Requires GTK 1..1.16 or higher  to use MP3x.  (www.gtk.org )
Newer Beta versions.    Frequently updated (see history) but not fully tested.   Any feedback is greatly appreciated.

MP3x binary (250 kb)

Binary version of MP3x.  This is the graphical analyzer without encoding capabilities.  It will analyze both .wav and .mp3 files.
  • Compiled on RedHat Linux 5.2 (Intel).
  • Requires  GTK 1.2.1 or higher.
