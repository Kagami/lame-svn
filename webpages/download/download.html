LAME - Source Code Download Page

NOTE: Before downloading any MP3 related software, please read this important message from the RIAA.

LAME is only distribued in source code form. For binaries and GUI based programs which can use LAME (or include fully licensed versions of LAME), check the LAME related links.

LAME source code is maintained with CVS, hosted on SourceForge.net If you have CVS, you can access this source code directly, see The LAME project on SourceForge.net. You can also download tarballs of the CVS repository, created daily, at http://cvs.sourceforge.net/cvstarballs/lame-cvsroot.tar.gz
