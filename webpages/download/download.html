LAME  Source Code Download Page

NOTE: Before downloading any MP3 related software, please read this important message from the RIAA.

For binaries and GUI based programs which can use LAME (or include LAME built in),  check the  LAME related links.

Click here for lame3.70tar.gz and newer beta versions.     LAME compiles on GNU/Linux, Unix, Windows, DOS, MacOS, BeOS, riskOS, AmigaOS, OS/2, FreeMiNT(Atari).


Beta versions are frequently updated (see history ) but not fully tested.   Any feedback is greatly appreciated.

LAME source code is maintained with CVS, hosted on SourceForge.net .   If you have CVS, you can access this source code directly, see  The LAME project on SourceForge.net.   You can also download tarballs of the CVS repository, created daily, at http://cvs.sourceforge.net/cvstarballs/lame-cvsroot.tar.gz
 

Mac users should check out DropMP3 Work is underway to merge the required Mac modifications (for the shared encoding library) into LAME.

Complete source code is also available from:

  • The not_lame encoder site
  • Bleeding edge versions from the sourceforge CVS repository are available from this site
  • The programmers corner of www.mp3-tech.org .
  • This ftp site.
  • The ZLURP ftp site.
  • Mac specific patches available from the MacLame site.   (down as of 4/00, but may return)
