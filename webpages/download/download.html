LAME  Download Page

NOTE: Before downloading any MP3 related software, please read this important message from the RIAA.

To find GUI based programs which can use LAME (or include LAME built in),  check the  LAME related links.

Binaries/DLL's of the LAME command line encoder may be available from the following sites (not affiliated with the LAME project): 

  • Linux & Unix:  binaries for stable releases of LAME are available from the not_lame encoder site
  • Windows and OS/2  :  LAME.EXE for Windows and OS/2 available from:   (Not)LAME for OS/2 and DOS
  • Windows:  From the Encoder/Windows section of   www.mp3-tech.org 
  • Windows:  Beta and Bleeding edge versions from this site in Russia!
  • Windows:  Bleeding edge versions compiled nightly from the sourceforge CVS repository are available here.
  • Windows:  From the  ZLURP ftp site.
  • Amiga:  For those of you that just wont give up on your Amiga, check The Amiga Alternative Audio Page

Complete source code distributions (no patching necessary) are available from:

  • The not_lame encoder site,
  • Bleeding edge versions from the sourceforge CVS repository are available from this site
  • The programmers corner of www.mp3-tech.org.  
  • This ftp site.
  • The ZLURP ftp site.

Mac: Source code and Mac specific patches available from the MacLame site.

  • Requires ISO demonstration source: dist10.tar (Univ. Hannover, MP3Tech,  other sites )
