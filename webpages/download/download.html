LAME w/MP3x  and GPSYCHO  (gzipped patch file, latest stable release)

Here is a script with directions for applying the patch ( applypatch  ). Requires ISO demonstration source: dist10.tar (Univ. Hannover, MP3Tech,  other sites )

This is a source code patch against the ISO demonstration source. It contains none of the original ISO encoder source code.  It  includes the LAME encoder patches, mpg123 for decoding, the GPSYCHO psy-model and the MP3x graphical analyzer.

 

Newer Beta versions.    Full source code - no patching necessary!

In LAME3.81beta and up, all ISO code has been removed.  These versions are distributed with the complete source code.     Beta versions are frequently updated (see history) but not fully tested.   Any feedback is greatly appreciated.
 

LAME source code is maintained with CVS, hosted on SourceForge.net.   For anonymous CVS access, go to the  the LAME project on SourceForge.net
