Free (speech) software which supports or uses LAME:

  • Grip:  (Linux) gtk-based cd-player, ripper and encoder.  Supports cddb, cdparanoia and LAME.
  • Krabber  (Linux) A KDE  ripper & encoder, can use LAME.
  • Mp3Maker  (Linux) A WindowMaker enhanced front end to cdda2wav/cdparanoia and lame/bladeenc.
  • ripperX  (Linux) GTK frontend for rippers and several encoders featuring CDDB support.
  • RaviLAME (Mac) Ripper & encocder.  Source code only, includes LAME.
  • DropMP3 (Mac)  includes LAME binaries.
  • CDex:   (Windows)   Ripper & encoder, includes LAME binaries (the Blade compatible dll)
  • LAMEX (Windows) An activex control for LAME, and a GUI.  Source code only, includes LAME.
  • RazorLame (Windows)  The  RazorBlade front end now supports LAME.
  • WinLama (Windows) Small frontend for LAME.EXE
  • LiveIce  Real time streaming of mp3s.  Works with IceCast.
  • Flash Forth A Flash-like development library

Other MP3 Encoders for Linux:

  • Takehiro Tominaga has done some amazing work on LAME recently (see  history).  His  versions are available here (source code only)    Fortunately for us,  Takehiro takes the time to fold many of his improvements back into LAME.
  • GOGO-no-coda  MP3 encoder.   They have forked LAME development at LAME 3.23beta.  Many key routines rewritten in assembly.  Uses the NASM assembler.
  • mp3enc3.1  from FhG.  The gold standard of encoders.  All other encoders strive to sound this good.  Demo version only encodes up to 30 seconds.
  • BladeEnc  One of the first free encoders for Linux.
