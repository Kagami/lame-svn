Free (speech) software which supports or uses LAME:

  • CDex:  ripper & encoder (for Microsoft Windows)
  • Grip:   gtk-based cd-player, ripper and encoder.  Supports cddb, cdparanoia and LAME.
  • Krabber  A KDE front-end ripper & encoder.
  • Mp3Maker  A WindowMaker enhanced front end to cdda2wav/cdparanoia and lame/bladeenc.
  • ripperX  GTK frontend for rippers and several encoders featuring CDDB support.
  • LiveIce  Real time streaming of mp3s.  Works with IceCast.

Free (beer) software which supports or uses LAME:

  • AutoLAMEr   Windows frontend for LAME.
  • LameBatch  Windows frontend for LAME.
  • RipChord   ripper and encoder for BeOS.
  • SecondSpin  ripper & encoder (for the Amiga)
  • ZLURP   CD-Ripper/Encoder/Tagger  (Windows, Mac, Linux)

Commecial software which supports or uses LAME:

  •  Easy CD-DA Extractor   Windows ripper/encoder

Commercial software to be avoided:

  • Proteron N2MP3.  This program may contain an uncredited ripoff of the LAME encoding engine in volation of the GPL.

The gold standard of encoders:

  • MP3Encoder  from FhG.  All other encoders strive to sound this good.  Demo version only encodes up to 30 seconds.

Other MP3 encoders:

  • BladeEnc  The most popular free encoder for Linux.  BladeEnc is a faster version of the ISO demonstration encoder with an improved user interface.
