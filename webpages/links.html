If you know how to use a command line: 

  • GOGO-no-coda  MP3 encoder.   They have forked LAME development at LAME 3.23beta.  Many key routines rewritten in assembly.  Requires the NASM assembler.
  • Takehiro Tominaga has done some amazing work on LAME recently (see  history).  His  versions are available here (source code only)    Fortunatly for us,  Takehiro takes the time to fold many of his improvements back into LAME.
  • Linux & Unix binaries for LAME are available from the not_lame encoder site.
  • LAME.EXE for Windows and OS/2 available from:   (Not)LAME for OS/2 and DOS (broken, as of 3/00)
  • LAME.EXE for Windows available from  mp3tech.org  (mp3tech.org is temporally located here)  as well as from Russia!
  • For those of you that just wont give up on your Amiga, check The Amiga Alternative Audio Page

Free (speech) software which supports or uses LAME:

  • CDex:   (Windows)   Ripper & encoder, includes LAME binaries (the Blade compatiable dll)
  • Grip:  (Linux) gtk-based cd-player, ripper and encoder.  Supports cddb, cdparanoia and LAME.
  • Krabber  (Linux) A KDE  ripper & encoder, can use LAME.
  • LAMEX (Windows) An activex control for LAME, and a GUI.  Source code only, includes LAME.
  • Mp3Maker  (Linux) A WindowMaker enhanced front end to cdda2wav/cdparanoia and lame/bladeenc.
  • RaviLAME (Mac) Ripper & encocder.  Source code only, includes LAME.
  • ripperX  (Linux) GTK frontend for rippers and several encoders featuring CDDB support.
  • LiveIce  Real time streaming of mp3s.  Works with IceCast.

Free (beer) software which supports or uses LAME:

  • AutoLAMEr   (Windows) frontend for LAME.EXE
  • LameBatch  (Windows) frontend for LAME.EXE
  • RipChord   (BeOS) Ripper and encoder.  Includes LAME binaries.  (link broken 3/00)
  • SecondSpin  (Amiga) ripper & encoder
  • ZLURP   (Windows, Mac, Linux) Ripper and encoder.  Includes LAME binaries.

Commecial software which supports or uses LAME:

  •  Easy CD-DA Extractor   Windows ripper/encoder, includes LAME binaries.

The gold standard of encoders:

  • MP3Encoder  from FhG.  All other encoders strive to sound this good.  Demo version only encodes up to 30 seconds.

Other MP3 encoders:

  • BladeEnc  One of the first free encoders for Linux.  BladeEnc is a faster version of the ISO demonstration encoder with an improved user interface.
