The LAME Project

LAME is an LGPL MP3 encoder. The Open source development model allowed to improve its quality and speed since 1999. It is now an highly evolved MP3 encoder, with quality and speed able to rival state of the art commercial encoders.

 

Latest LAME release : v3.96 April 2004

Technical Details (MP3 Encoding and LAME)

  • Join the mp3 encoding mailing list   and see the archives (older archives:   October 2000 - Sept 2001   and  before October 2000 )
  • See the mp3 forums on Mp3-tech and HydrogenAudio
  • IRC:    #mp3encoder, on Open Projects Network



LAME 3.xx uses software from:

  • Decoding engine (for the frame analyzer):  mpglib, from the mpg123 project.  Mpg123 is an excellent  MPEG player.  Mpglib is released under the GPL.
  • Input from almost any type of sound file now handled by the GPL'd libsndfile .
